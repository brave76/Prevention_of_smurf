
# Prevention of Smurf Attack
There are some ways of Prevention of smurf attack.

## Disable Broadcast 
Disable Broadcast on router. When we Disable Broadcast on router any host on network can't braodcast a message.

## Disable Ping
Disable ping response on your system.

## Disallow Ping on firewall
Disable ping response on  your firewall.

## Block Broadcast Traffic
Block braodcast where comming from outer network.

## Block Range of IPs
Block specific Range of IPs those are performing invalid activity.

## Network Monitoring
Monitor your network with IDS.

## Deploy Next-Gen Antivirus / WPA
If want to prevent your site Deploy lest next generation antivirus and web application firewall

## Built Redundancy
Deploy more then one servers. So no one can perform this attack on your self. Buy more bandwidth for managing high traffic.

## Threshold Values
Set Threshold Values. when a requests touch to a specific Threshold value. Then that ip block automatically.

## Network Ingress Filtering
It will block requests comming from untrusted ips.

## Packet Filtering
Write fire roles for checking every Packet comming from outer network.
